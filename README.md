# Dansish compiler
Why not code in your native language?

## Example
main.dk:
```
Offentlig Funktion Hovedsagelig tilbagegiver heltal.
Goddag.
    Lad C være et heltal.
    Sæt C til kald LægSammen 10 , 10,0 .
    Kald Print C.
    Tilbagegiv 0.
Farvel.

Offentlig Funktion LægSammen
    tager A som heltal,
    tager B som heltal,
    tilbagegiver heltal.
Goddag.
    Tilbagegiv A plus B.
Farvel.
```

To compile and run main.dk:
```
dansk run ./main.dk
```

## Build
From an MSVC command promt:
```
build.bat
```
